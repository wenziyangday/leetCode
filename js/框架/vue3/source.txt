              }

              reactiveMap: WeakMap


          createReactiveObject()
            target: Target
            isReadonly: boolean
            baseHandlers: ProxyHandler
            collectionHandlers: ProxyHandler
            proxyMap: WeakMap
              必须以对象作为键
              键为弱键，即作为键的对象被删除或者被重新赋值则键值对会同时消失
              由于键是弱键，导致无法判断此时此刻，彼时彼刻任一键值对是否存在，导致没有迭代器
                WeakMap这样的特性意味着当某个target消失或者重新赋值后
                代理对象所在的键值对在内存中也被自动清除了
                极大的提高了内存空间的使用效率
                数据量积累到一定程度后，会对性能也有极大优化

            -处理逻辑
              !isObject(target) return target

              ProxyMap.get(target)  return  ProxyMap.get(target) 说明target 已经是一个proxy，并存在与weakMap中
              getTargetType(target) === INVALID return target
              return new Proxy(target, handler)
