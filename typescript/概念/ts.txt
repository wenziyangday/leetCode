unknown 和 any
any 跳过类型检察，而unknown作为一个未知类型，是不允许其访问属性，也不允许赋值给其他类型明确的对象的

Void 类型： void
  表示函数没有返回值 或者值为 undefined
  对于声明为void类型的值只能赋值为undefined或者null

Never类型：never
  表示永远不会出现的值的类型
  Never类型用于处理异常或处理程序中的无限循环等无法正常结束的情况

类型断言 as
  vs 类型转换
  vs 类型声明
  vs 泛型

默认直接导出  function、class、interface

type interface 区别
  几乎所有的interface功能 都能通过type 实现
  区别：
    接口扩展通过extends、type扩展字段是通过&
    接口可以重复声明相同名称的（合并）、type对于相同名称的类型声明会出现报错

枚举：
  手动赋值(可以使用位运算)
  常数枚举 const enum
  外部枚举 declare enum
  数字枚举建议通过手动赋值 从1开始避免不必要的判断false （0）

接口（Interfaces）：
  不同类之间公有的属性或方法，可以抽象成一个接口。
  接口可以被类实现（implements）。一个类只能继承自另一个类，但是可以实现多个接口

  类实现接口、接口继承接口、接口继承类（类的声明处理创建一个类外，还会创建一个同名的类型（实例的类型））

泛型：
  在定义函数、接口、类的时候，不预先指定具体的类型，而在使用的时候再指定类型的一种特性。

声明合并：
  同名函数（重载）：
  同名接口（相同名称，不同类型的属性 会报错）

重写：
  属性重写
  方法重写

  通过新的赋值或者实现从而实现 新的行为或者功能
